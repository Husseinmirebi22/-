# 7 – البرامج والملفات التنفيذية وتطوير الأنظمة (Executables and System Development)

**الوصف:** تشمل هذه المجموعة صيغ ملفات البرامج القابلة للتنفيذ المباشر، والمكتبات المشتركة، وملفات النظام والتعريفات، وصور نظام التشغيل.

---

## .axf

- **النوع:** executable
- **الخصائص:** ثنائي، ARM Executable Format (ELF معدل)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** ملفات تشغيل في المعالجات المدمجة (ARM)
- **البرامج التي تفتحه:** ARM Development Studio
- *_تمت الإضافة بواسطة المساعد (سياق تشغيل الأنظمة)._*

---

## .bat

- **النوع:** script
- **الخصائص:** نصي، ملف دفعي لويندوز (Batch File)
- **القابلية للنص:** نصي
- **الاستخدام الشائع:** تشغيل سلسلة من تعليمات سطر الأوامر في CMD لويندوز
- **البرامج التي تفتحه:** cmd.exe، أي محرر نصوص

---

## .bin

- **النوع:** executable_binary
- **الخصائص:** ثنائي، ملف تنفيذي خام لويندوز أو يونكس
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** برامج ذاتية التشغيل منخفضة المستوى، مثبتات لينكس، والبرامج الثابتة
- **البرامج التي تفتحه:** موجه الأوامر، المحاكيات، Hex Editor

---

## .cmd

- **النوع:** script
- **الخصائص:** نصي، ملف تنفيذ الأوامر لويندوز NT
- **القابلية للنص:** نصي
- **الاستخدام الشائع:** تشغيل سكريبتات متطورة على سطر الأوامر البديل لـ DOS
- **البرامج التي تفتحه:** cmd.exe، أي محرر نصوص

---

## .com

- **النوع:** executable
- **الخصائص:** ثنائي، Command File (DOS/16-bit Windows)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** برامج كلاسيكية لا تتجاوز 64 كيلوبايت
- **البرامج التي تفتحه:** MS-DOS (عبر DosBox أو موجه 32-بت)

---

## .dll

- **النوع:** library_shared
- **الخصائص:** ثنائي، Dynamic Link Library (مكتبة ويندوز مشتركة)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** توفير كود وميزات قابلة لإعادة الاستخدام من قبل برامج متعددة
- **البرامج التي تفتحه:** نظام التشغيل ويندوز، Dependency Walker

---

## .drv

- **النوع:** driver
- **الخصائص:** ثنائي، ملف تعريف جهاز لويندوز (Driver)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** برمجيات منخفضة المستوى للتواصل بين ويندوز وقطع العتاد
- **البرامج التي تفتحه:** نظام التشغيل ويندوز
- *_تمت الإضافة بواسطة المساعد._*

---

## .dylib

- **النوع:** library_shared
- **الخصائص:** ثنائي، Dynamic Library (macOS)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** مكتبات برمجية مشتركة ومستدعية في نظام ماك
- **البرامج التي تفتحه:** macOS Loader

---

## .elf

- **النوع:** executable
- **الخصائص:** ثنائي، Executable and Linkable Format (Linux / Embedded)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** البرامج التنفيذية والمكتبات المشتركة في معظم أنظمة لينكس واللوحات الإلكترونية
- **البرامج التي تفتحه:** نظام التشغيل لينكس، readelf

---

## .exe

- **النوع:** executable
- **الخصائص:** ثنائي، Portable Executable (Windows)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** تثبيت وتشغيل برامج وتطبيقات نظام ويندوز (32/64-بت)
- **البرامج التي تفتحه:** نظام التشغيل ويندوز

---

## .gadget

- **النوع:** widget
- **الخصائص:** ثنائي (ZIP)، أداة مصغرة لويندوز فيستا و7
- **القابلية للنص:** ثنائي (HTML/JS داخل ZIP)
- **الاستخدام الشائع:** لوحات الطقس والساعات على سطح المكتب الكلاسيكي
- **البرامج التي تفتحه:** Windows Sidebar (قديم)

---

## .ipa

- **النوع:** application_archive
- **الخصائص:** ثنائي (ZIP)، حزمة تطبيق iOS
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** ملفات التوزيع وتنزيل الألعاب والبرامج على منتجات آبل
- **البرامج التي تفتحه:** نظام iOS، iTunes، Xcode
- *_تمت الإضافة بواسطة المساعد (سياق التوزيع والتطبيقات)._*

---

## .msi

- **النوع:** installer
- **الخصائص:** ثنائي، Microsoft Installer Database
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** تثبيت البرمجيات المهيكلة والأمنية في ويندوز
- **البرامج التي تفتحه:** Windows Installer Engine

---

## .msp

- **النوع:** installer_patch
- **الخصائص:** ثنائي، Windows Installer Patch
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** توزيع تحديثات ورقوع برامج معينة في بيئات الشركات
- **البرامج التي تفتحه:** Windows Installer Engine
- *_تمت الإضافة بواسطة المساعد._*

---

## .o

- **النوع:** object_file
- **الخصائص:** ثنائي، Object file (ناتج المترجم)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** كود مجمع غير مربوط (بانتظار Linker ليصبح exe أو elf)
- **البرامج التي تفتحه:** GCC، Clang، MSVC Linker

---

## .out

- **النوع:** executable
- **الخصائص:** ثنائي، ملف تنفيذي عام (يونكس قديماً) أو مخرجات تصحيح
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** البرامج السريعة ومخرجات التجميع في يونكس وعلم الحوسبة
- **البرامج التي تفتحه:** نظام التشغيل يونكس/لينكس

---

## .ps1

- **النوع:** script
- **الخصائص:** نصي، PowerShell script (ويندوز ولينكس)
- **القابلية للنص:** نصي
- **الاستخدام الشائع:** أتمتة مهام الإدارة والتحكم المعقدة في أنظمة ويندوز والشبكات
- **البرامج التي تفتحه:** PowerShell Console، VS Code، أي محرر نصوص

---

## .scr

- **النوع:** screen_saver
- **الخصائص:** ثنائي، حافظة شاشة ويندوز (تنفيذي متنكر)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** شاشات توقف تفاعلية لويندوز (انتبه: قد تُستغل في الهجمات لأنها ملفات exe فعلياً)
- **البرامج التي تفتحه:** Windows desktop system

---

## .sh

- **النوع:** script
- **الخصائص:** نصي، Shell script (Bash/Sh)
- **القابلية للنص:** نصي
- **الاستخدام الشائع:** أتمتة الأنظمة وتنزيل وإعداد البرمجيات على سيرفرات لينكس وماك
- **البرامج التي تفتحه:** Bash Terminal، أي محرر نصوص

---

## .so

- **النوع:** library_shared
- **الخصائص:** ثنائي، Shared Object (ديناميكية لينكس/أندرويد)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** توفير أكواد مشتركة ومحركات تشغيل الألعاب بكفاءة عالية بلغات C/C++
- **البرامج التي تفتحه:** Linux dynamic linker, Android loader

---

## .sys

- **النوع:** driver
- **الخصائص:** ثنائي، ملف نظام ويندوز أو تعريف عتاد منخفض المستوى
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** أكواد تشغيل النواة (Kernel mode drivers) وإدخال وإخراج عتاد ويندوز
- **البرامج التي تفتحه:** Windows Kernel Engine

---

## .vb

- **النوع:** script
- **الخصائص:** نصي، Visual Basic script
- **القابلية للنص:** نصي
- **الاستخدام الشائع:** سكريبتات أتمتة وسير عمل خفيفة
- **البرامج التي تفتحه:** Windows Script Host، أي محرر نصوص

---

## .vbs

- **النوع:** script
- **الخصائص:** نصي، Visual Basic Scripting Edition
- **القابلية للنص:** نصي
- **الاستخدام الشائع:** برمجة سكريبتات ويندوز التفاعلية (قديمة وتستخدم في الأتمتة الموروثة)
- **البرامج التي تفتحه:** Windows Script Host، أي محرر نصوص
