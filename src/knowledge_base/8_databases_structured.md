# 8 – قواعد البيانات والملفات المهيكلة (Databases and Structured Files)

**الوصف:** تشمل هذه المجموعة صيغ قواعد البيانات العلائقية وغير العلائقية، والمحركات المحلية، وتنسيقات حفظ وتبادل السجلات المهيكلة رقمياً.

---

## .accdb

- **النوع:** relational_db
- **الخصائص:** ثنائي، Microsoft Access Database (حديث)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** تخزين وتصميم قواعد بيانات وجداول ونماذج مايكروسوفت أكسس
- **البرامج التي تفتحه:** Microsoft Access، مكتبات ACE

---

## .csv

- **النوع:** structured_table
- **الخصائص:** نصي، Comma-Separated Values
- **القابلية للنص:** نصي
- **الاستخدام الشائع:** تصدير السجلات وتبادل قواعد البيانات البسيطة والتعامل مع Pandas ودراسة البيانات
- **البرامج التي تفتحه:** Microsoft Excel، VS Code، أي محرر نصوص

---

## .db

- **النوع:** relational_db_local
- **الخصائص:** ثنائي، ملف قاعدة بيانات محلي عام (غالباً SQLite)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** حفظ وتخزين تفضيلات وسجلات التطبيقات وهواتف أندرويد وآيفون محلياً
- **البرامج التي تفتحه:** DB Browser for SQLite، SQLite CLI

---

## .dbf

- **النوع:** relational_db_flat
- **الخصائص:** ثنائي، dBASE database
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** قواعد البيانات المسطحة الموروثة، وجداول بيانات نظم المعلومات الجغرافية (GIS Shapefiles)
- **البرامج التي تفتحه:** dBase, Microsoft Excel، QGIS

---

## .dbx

- **النوع:** email_db
- **الخصائص:** ثنائي، Outlook Express Mailbox
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** تخزين المجلدات والرسائل في بريد Outlook Express القديم
- **البرامج التي تفتحه:** Outlook Express، ملحقات تحويل DBX
- *_تمت الإضافة بواسطة المساعد._*

---

## .fdb

- **النوع:** relational_db
- **الخصائص:** ثنائي، Firebird database
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** قواعد بيانات علائقية محلية وموزعة للمنشآت المتوسطة وصاحبة السجلات الموروثة
- **البرامج التي تفتحه:** Firebird Server، FlameRobin

---

## .gdb

- **النوع:** geodatabase
- **الخصائص:** ثنائي (مجلد)، Esri File Geodatabase
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** تخزين قواعد البيانات ونظم الجغرافيا (GIS datasets) في Esri
- **البرامج التي تفتحه:** ArcGIS Pro، QGIS
- *_تمت الإضافة بواسطة المساعد._*

---

## .json

- **النوع:** structured_data
- **الخصائص:** نصي، JavaScript Object Notation (مهيكل هرمياً)
- **القابلية للنص:** نصي
- **الاستخدام الشائع:** حفظ وتبادل بيانات الويب، وتفضيلات التطبيقات وقواعد بيانات NoSQL (مثل MongoDB)
- **البرامج التي تفتحه:** أي متصفح، أي كاتب نصوص، VS Code

---

## .jsonl

- **النوع:** structured_data_lines
- **الخصائص:** نصي، JSON Lines (سجلات متتالية بسطور مستقلة)
- **القابلية للنص:** نصي
- **الاستخدام الشائع:** تغذية بيانات الذكاء الاصطناعي (Fine-tuning)، وسجلات الـ Cloud Logging الضخمة
- **البرامج التي تفتحه:** أي محرر نصوص، سكريبتات Python

---

## .ldb

- **النوع:** database_lock
- **الخصائص:** ثنائي، ملف أقفال Access (Microsoft Access Lock File)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** منع الكتابة المتزامنة والتعارض عند فتح قاعدة بيانات Access من قبل مستخدمين متعددين
- **البرامج التي تفتحه:** يفتحه تلقائياً محرك Jet/Access

---

## .mdb

- **النوع:** relational_db
- **الخصائص:** ثنائي، Microsoft Access Database (الموروث، محرك Jet)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** قواعد بيانات تطبيقات سطح المكتب القديمة وتصميم جداول أكسس التاريخية
- **البرامج التي تفتحه:** Microsoft Access

---

## .ndjson

- **النوع:** structured_data
- **الخصائص:** نصي، Newline Delimited JSON (معادل لـ JSONL)
- **القابلية للنص:** نصي
- **الاستخدام الشائع:** نقل جزيئات البيانات وسجلاتها الكبيرة بكفاءة عالية وبدون قراءتها كاملة في الذاكرة
- **البرامج التي تفتحه:** أي محرر نصوص، سكريبتات Python
- *_تمت الإضافة بواسطة المساعد._*

---

## .nsf

- **النوع:** document_db
- **الخصائص:** ثنائي، Lotus Notes Database
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** تخزين بريد وسجلات ومفكرات خوادم IBM/HCL Domino
- **البرامج التي تفتحه:** HCL Notes / Domino

---

## .odf

- **النوع:** formula_structured
- **الخصائص:** ثنائي (ZIP)، OpenDocument Formula (مستند المعادلات)
- **القابلية للنص:** ثنائي (XML داخل ZIP)
- **الاستخدام الشائع:** حفظ الصيغ الرياضية والمعادلات الرقمية في حزم أوفيس المفتوحة
- **البرامج التي تفتحه:** LibreOffice Writer, Math

---

## .pdb

- **النوع:** database_palm
- **الخصائص:** ثنائي، Palm OS Database
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** تخزين قواعد بيانات صغيرة في هواتف وأجهزة كفية سابقة (Palm Pilot)
- **البرامج التي تفتحه:** Palm Desktop, Calibre (للكتب المحولة)

---

## .sql

- **النوع:** script_database
- **الخصائص:** نصي، تعليمات Structured Query Language
- **القابلية للنص:** نصي
- **الاستخدام الشائع:** تصدير (Dump) قواعد البيانات الكبرى (PostgreSQL, MySQL, SQLite) وتصميم الجداول وإدراج القيم
- **البرامج التي تفتحه:** pgAdmin، MySQL Workbench، أي محرر نصوص

---

## .sqlite

- **النوع:** relational_db_local
- **الخصائص:** ثنائي، SQLite Database
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** قواعد البيانات المدمجة والخفيفة داخل التطبيقات للأجهزة الذكية وسطح المكتب
- **البرامج التي تفتحه:** DB Browser for SQLite، DBeaver

---

## .sqlite3

- **النوع:** relational_db_local
- **الخصائص:** ثنائي، SQLite v3 Database (تأكيد الإصدار)
- **القابلية للنص:** ثنائي
- **الاستخدام الشائع:** المعيار الرئيسي لقواعد البيانات المحلية والسريعة في متصفحات الويب والتطبيقات المحلية
- **البرامج التي تفتحه:** SQLite3 engine، DB Browser for SQLite

---

## .tsv

- **النوع:** structured_table
- **الخصائص:** نصي، Tab-Separated Values
- **القابلية للنص:** نصي
- **الاستخدام الشائع:** فرز البيانات الكبيرة وعلم المعلومات البيومترية والمحاسبة بدون تداخل غبي للفواصل مع علامات التحويل
- **البرامج التي تفتحه:** Microsoft Excel، Python (Pandas)، أي كاتب نصوص

---

## .xml

- **النوع:** structured_markup
- **الخصائص:** نصي، Extensible Markup Language
- **القابلية للنص:** نصي
- **الاستخدام الشائع:** تبادل البيانات المهيكلة بين الأنظمة المختلفة وتعريف واجهات وبرتوكولات الشبكة (SOAP)
- **البرامج التي تفتحه:** أي متصفح، أي كاتب نصوص، محررات XML الشهيرة

---

## .yaml

- **النوع:** structured_config
- **الخصائص:** نصي، YAML Ain't Markup Language (اعتماد مسافات بادئة بادية)
- **القابلية للنص:** نصي
- **الاستخدام الشائع:** إعداد وكلاء ومشاريع الذكاء الاصطناعي، البنية التحتية ككود (Terraform, Docker Compose)، وإعداد المشاريع البرمجية
- **البرامج التي تفتحه:** VS Code، أي محرر نصوص وقراءة YAML
